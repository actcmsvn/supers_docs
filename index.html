Cám bạn đã mua code superS.

Cài đặt cho ứng dụng người dùng Sau khi cài đặt, bạn có thể áp dụng điều tương tự trên ứng dụng Giao hàng nhanh chỉ cần bỏ qua cài đặt Plugin

Nó rất đơn giản để cài đặt. Chỉ cần làm theo bước dưới đây (linh hoạt giữa các bước) và bạn đã sẵn sàng để bắt đầu.

  • Cài đặt cordova,Nodejs & ionic phiên bản mới nhất
  • Quan trọng Cài đặt phiên bản mới nhất của dự án trống ionic. Thay thế các tệp (resources, src, ionic.config.json, config.xml) đã tải xuống này trong thư mục gốc của dự án của bạn.

    • Cài đặt Platform & Plugin Bắt buộc

  • thêm Platform thiết bị androidionic cordova platform add android
  • thêm Platform thiết bị iosionic cordova platform add ios
  • Cài đặt các Plugins bắt buộc:
  • OneSignal
  • Geolocation
  • Geocoder
  • PayPal
  • Stripe Payment

    • Chạy thử ứng dụng

    Bây giờ bạn đã sẵn sàng để chạy ứng dụng trong trình duyệt

  • Chạy lệnh này trong terminal từ thư mục dự án của bạn ionic serve
  • Nếu yêu cầu cập nhật node-sass thì hãy chạy lệnh này npm rebuild node-sass
  • Thay đổi api URL trong src/app/service/server.server.ts với URL thay đổi và thêm /api/ phía sau đường dẫn url ví dụ https://www.abc.com/api/
  • Ứng dụng đã sẵn sàng để chạy
  • Nếu bạn muốn sử dụng paypal, hãy thay đổi ID khách hàng trong cài đặt ở trang quản trị & lưu thay đổi PayPalEnvironmentSandbox thành PayPalEnvironmentProduction trong file src/app/checkout/checkout.ts

Thay đổi Giao diện ứng dụng

  • Bạn có thể thay đổi biến màu trong file src/theme/veriable.css
  • để thay đổi icon ứng dụng và hình splash lúc mở ứng dụng, bạn có thể thay file icon.png (1024x1024) & splash.png (2732x2732) trong thư mục resources và sử dụng câu lệnh ionic cordova resources --icon và ionic cordova resources --splash
  • Để thiết lập google map, bạn phải nhập Khóa API google map của mình vào file src/index.html trong app & file local/resources/views/admin/user/google.blade.php trong web
  • Để thiết lập thông báo đẩy onesignal, thay đổi API trong src/app/app.component.spec.ts & trong file web local/app/http/controller/controller.php

Build APK

  • Để build thành ứng dụng cho androidionic cordova build android --prod --release
  • Tạo signing key keytool -genkey -v -keystore my-release-key.keystore -alias alias_name -keyalg RSA -keysize 2048 -validity 10000
  • Để ký ứng dụng với keystore jarsigner -verbose -sigalg SHA1withRSA -digestalg SHA1 -keystore my-release-key.keystore app-release-unsigned.apk alias_name
  • Để đăng ký ứng dụng để xuất bản trong playstore zipalign -v 4 superS-release-unsigned.apk superS.apk

Cài đặt API

  • Tải lên các tệp API lên máy chủ của bạn (yêu cầu phiên bản php >= 7.1)
  • Thay đổi chi tiết Cơ sở dữ liệu trong file .env file nằm trong thư mục local
  • Từ file.env bạn cũng thay đổi chi tiết SMTP email.
  • Thêm Google API KEY vào local/resources/view/admin/user/google.blade.php ở cuối file để lấy vị trí của google

    • Quá tuyệt! Bạn đã sẵn sàng. Chúc vui vẻ
